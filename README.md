golangの練習の一環として作成しているtwitter botです．  
heroku app上で動いています．

# 現在の機能
- 4時間おきに，DBからランダムにtextを取得してツイートしています  
- 時計の短針と連動して，アイコンが回転します(更新：1時間ごと)
